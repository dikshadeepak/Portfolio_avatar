# 🎙️ AI Talking Portfolio

An interactive AI-powered developer portfolio that not only showcases my skills and projects but also communicates through voice interactions. Built using modern web technologies with beautiful animations, responsive UI, and immersive user experience.

---

## 🌟 Overview

This project is more than just a personal portfolio.

It combines modern frontend development with AI-inspired interaction to create a portfolio that feels alive. Visitors can explore my projects, skills, education, and achievements while interacting with an intelligent talking interface.

The portfolio is fully responsive, highly animated, and designed to provide a unique user experience.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/yourusername/talking-portfolio.git
```

Move into the project directory

```bash
cd talking-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
